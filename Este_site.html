Irei destacar rapidamente a seguir, as etapas para criar este projeto. Para mais informações do código, entre em contato ou veja meu GitHub na aba "contatos". Posteriormente, com a estrutura formada, desenvolvi em CSS os efeitos para links como por exemplo as opções do menu que mudam de cor ao ocorrer interação com o usuário.

Por fim, adicionei funções em JavaScript para termos a opção "ler mais" e " ler menos" presentes nas postagens da página inicial. Passou por lá e não percebeu? Sem problemas, pode voltar lá e ver que eu espero! :) Cada botão tem uma função específica que é ativada ao clicar. A função basicamente observa através das variáveis criadas, se a parte maior do texto está visível ou não, caso esteja, o script esconde o texto e muda o botão para "ler mais". Também é válido o mesmo processo ao inverso.
